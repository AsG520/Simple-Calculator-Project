How it Works

Requirements
- IDE (e.g., VS Code, Eclipse, or IntelliJ).
- HTML (HyperText Markup Language).
- CSS (Cascading Style Sheets).
- JS (JavaScript).
- Live Reloading or Hot Reloading (e.g., Live Server, and etc).

How to Run
i) Copy the code into a Java file named "FrictionBox.java".
ii) Copy the HTMl code into a HTML file named "index.html".
iii) Copy the CSS code into a CSS file named "style.css".
iv) Copy the JS code into a JS file named "function.js".
v) Make sure the files are in the same directory, or else program will fail to operate. 
vi) Run the program using your Live Reloading or Hot Reloading downloaded extension.
